# Select Star SQL (Traducción al Español)

Repositorio con la traducción al español de `selectstarsql.com`.

## Desarrollo

La estructura del código es la misma para un sitio construido con Jekyll. Todas las páginas están en formato Markdown en la carpeta raíz y se generan con Jekyll.

Para servir una versión local:

```pwsh
jekyll serve
```
